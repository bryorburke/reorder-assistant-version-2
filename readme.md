Baseline code for reordering a product depending on current stock levels, desired stock levels, and anticipated demand
Technologies Used
JavaScript: Used for logic processing, including mathematical modeling for safety buffers and reorder triggers.

HTML5: Structured the dashboard layout to display inventory metrics.

CSS3: Applied a professional "Corporate Dashboard" aesthetic using CSS Grid and Custom Properties (CSS Variables).

Google Fonts & Iconography: Enhanced readability and visual cues for status alerts.


How to Run the Project
Clone or Download: Save the project files (index.html, style.css, and script.js) into a single folder.

Open in Browser: Right-click the index.html file and select "Open with" followed by your preferred web browser (Chrome, Firefox, or Edge).

View Logic: To see the calculations in real-time, right-click anywhere on the page, select Inspect, and navigate to the Console tab to view the logic output alongside the visual dashboard.

Building this project I learned several insights on dynamic vs static logic, risk management principles, data visualization, and creating clean code.
